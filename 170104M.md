### Smart Contracts

Smart contract is a program stored inside the block chain. It runs when predetermined coditions are met. Normally smart contracts are used to automate the execution of an agreement that makes all the participants immediately become certain of the outcome. It avoids intermediary's involvements and time losses.

### Consensus Algorithms in blockchain

To confirm every transaction of the blockchain to be secured and verified, consensus protocol is used. With consesnsus algorithm all the peers of the blockchain network reach a common agreement about the state of the distributed ledger. There are several consensus algorithms namely: proof of work, proof of stake, proof of burn, etc.
